# 🎨 Creative Portfolio (Work in Progress)

This is a small project I’m working on while exploring web design and interactive UI development. It’s a starting point for my personal portfolio — currently focused on building a creative landing page using HTML, CSS, and a 3D background powered by [Spline](https://spline.design).

I’m just experimenting right now, but I plan to add more sections and different landing pages soon. Stay tuned!

---

## 🧪 Tech Stack

- **HTML5**
- **CSS3** (with animations and media queries)
- **JavaScript** (typing effect)
- **Spline** for embedded 3D visuals
- **Font Awesome** for icons

---

## 🧭 What's Inside?

- A clean and responsive layout
- Typing animation for name and role
- A 3D animated background using Spline
- Social media links and a button for downloading CV (placeholder)
